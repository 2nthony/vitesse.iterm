<h1 align="center">Vitesse theme for <a href="https://iterm2.com">iTerm2</a></h1>

<p align="center">
Vitesse dark only
</p>

## Usage

1. Open iTerm2 > Settings > Profiles
2. Select your profile > Colors > Color Presets > Import...
3. Select `vitesse.itermcolors`
4. Color Presets > `Vitesse`
